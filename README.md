# 👋 Centre d'apprentissage IT

Bienvenue ! Ce dépôt est mon carnet de bord pour apprendre l'infrastructure IT
**en autodidacte**, à mon rythme, sans centre de formation. Chaque domaine devient
un **module** qui synthétise la théorie et propose des labos reproductibles chez soi.

**➡️ [Ouvrir le portail](https://marvin32165.github.io/centre-apprentissage-it/)**

## Ce qu'on trouve ici

- 🖥️ **Windows Server** — AD, DNS/DHCP, DFS, sécurité, Core
- 💻 **PowerShell** — scripts, labo Hyper-V, GMSA
- 🐧 **Linux (Debian)** — terminal, permissions, réseau, APT, SSH, systemd
- 🌐 **Réseau & CCNA** — OSI, VLAN, routage, subnetting, NAT, ACL
- 🛡️ **Cybersécurité** — hacking éthique, pentest, hardening (usage légal/défensif uniquement)
- 📦 **Proxmox** — VM/LXC, ZFS, snapshots, cluster, HA
- 🏠 **Homelab & self-hosting** — Docker, reverse proxy, Pi-hole, backup
- ✉️ **Exchange / PKI**, 🖇️ **SharePoint**, 🖥️ **RDS**

Tout est **100 % statique** (HTML/CSS/JS, aucune dépendance) — il suffit d'ouvrir
`index.html` dans un navigateur pour naviguer entre les modules.
